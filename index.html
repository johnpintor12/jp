"Skip Response" Automation

This automation is designated as **"Skip Response."**

This means that if a customer's message matches this intent, the automation takes **no action**. It does not send a reply, tag the ticket, or route it.

The system simply logs that the match occurred, which helps to train and improve the AI's learning and accuracy over time.
